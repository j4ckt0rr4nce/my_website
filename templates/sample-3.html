Python program na nájdenie anagramu

V tejto ukážke sa pozrieme na to, ako môžeme zistiť, či sú dané reťazce anagramy. Najprv by sme sa však mali zoznámiť s konceptom anagramu.

Anagram je stav, keď je jeden reťazec alebo číslo usporiadané tak, kde každý znak preusporiadaného reťazca alebo čísla musí byť súčasťou iného reťazca alebo čísla. Inými slovami, reťazec môžeme nazvať anagramom iného reťazca vtedy, ak je druhý reťazec jednoduchým preskupením prvého reťazca. Napríklad python a yphton sú anagramy; Java a avaJ sú taktiež anagramy.

Poďme sa pozrieť na popis problému skôr ako ho vyriešime.

  • Získame vstupy reťazcov od používateľa a uložíme ich do samostatných premenných.
  • Na vytriedenie oboch reťazcov do listov použijeme metódu sort().
  • Skontrolujte oba listy, či tvoria anagram alebo nie.
  • Vytlačíme výsledok.
  • Ukončíme program.
def check_for_anagram(string1,string2):

    # Convert string into lists  
    list1 = list(string1)  
    list2 = list(string2) 

    # Sort the list value  
    list1.sort()  
    list2.sort()  
  
    index = 0  
    matches = True  
  
    while index < len(string1) and matches:  
        if list1[index]==list2[index]:  
            index = index + 1 
        else:  
            matches = False  
  
    return matches  
  
print(check_for_anagram('heart','earth'))
# Output
True

Vo vyššie uvedenom kóde sme deklarovali metódu check_for_anagram(), ktorá ako argument berie dva reťazce. Tieto reťazce sa prevedú do listu, v ktorom sa zoradia. Potom sme definovali premennú index a priradili sme jej nulu. V každej iterácii cyklu while je dĺžka reťazca porovnaná s hodnotou index listu. Každý prvok oboch listov sa navzájom porovnáva a zvýši hodnotu index o jedna. Akonáhle bude hodnota index väčšia ako dĺžka reťazca, cyklus sa ukončí a vráti hodnotu True; v opačnom prípade sa vráti False.

Existuje niekoľko techník a príkladov, ktoré môžeme použiť na nájdenie anagramu v Pythone. Nižšie sa na niekoľko z nich pozrieme.

Technika počítadla

Pri tejto technike zrátame počet znakov v oboch daných reťazcoch. Ak sa počet znakov daného reťazca zhoduje s iným reťazcom, potom je zodpovedajúcim slovom anagram druhého slova.

Pozrime sa na nasledujúci príklad.

from collections import Counter, defaultdict 

def check_for_anagram(strings):  
    ddict = defaultdict(list)  

    for i in strings:
        i_count = tuple(Counter(i).items())
        ddict[i_count].append(i)

    return list(ddict.values())  

strings = ("heart","earth")  

print(check_for_anagram(strings))
# Output
[['heart'], ['earth']]

Importovali sme modul collections a jeho metódy Counter a defaultdict na vyhľadanie anagramov reťazcov vo vyššie uvedenom kóde. Definovali sme metódu check_for_anagram() na spočítanie a zaznamenanie každého znaku pomocou našej funkcie. Počet každého znaku je zapísaný v liste na zodpovedajúcom mieste mieste. Tento proces sa vykonáva pre všetky znaky v prvom ako aj v druhom reťazci. Ak je počet znakov oboch reťazcov zhodný, znamená to, že naŠe reťazce sú anagramy.

Zoraďovacia technika

Pri tejto technike sú oba reťazce zoradené a funkcia kontroluje, či sa obe hodnoty navzájom zhodujú. Poďme sa pozrieť na nasledujúci príklad.

def check_for_anagram(string1, string2):  

    # Strings are sorted and check whether both are matching or not  
    if(sorted(string1) == sorted(string2)): 
        print("Both strings are an Anagram.")  
    else:  
        print("Both strings are not an Anagram.")  
         
word1 ="heart"  
word2 ="earth"  

print( "String-1 : ", word1 )
print( "String-2 : ", word2 )

check_for_anagram(word1, word2)
# Output
String-1 :  heart
String-2 :  earth
Both strings are an Anagram.

Vo vyššie uvedenom kóde sme definovali metódu check_for_anagram() a zadali dva reťazce ako parametre. V metóde check_for_anagram() sme reťazce uložili do konkrétnych premenných. Po zoradení sme porovnali každý reťazec. Ak sa porovnanie medzi reťazcami zhoduje, dané reťazce tvoria anagram; v opačnom prípade nám funcia vypíše, že reťazce anagram netvoria. Táto metóda je pomerne jednoduchá a účinná. Znižuje zložitosť kódu vo význačnej miere.

Reverzná kontrola anagramu

Túto techniku môžeme uplatniť nasledovne.

random_list = ["pizza", "cheeseburger", "now", "won"]
anagrams = {}  
for x in random_list:  
    reverse_word = x[::-1]
    if reverse_word in random_list:  
         anagrams[x] = (random_list.pop(random_list.index(reverse_word)))  
         print(anagrams)
# Output
{'now': 'won'}

Vo vyššie uvedenom kóde sme použili techniku na porovnanie anagramov pomocou reverzného reťazca. Vytvorili sme si dva rôzne reťazce. Táto technika je podobná palindrómom, kde sme jeden reťazec obrátili a porovnali s ostatnými reťazcami. Ak sa zhodujú, reťazce vytvorili anagram; ak sa nezhodujú, anagramy nebudú vypísané.

Technika overenia pozície

Pri tejto metóde sa na vyhľadanie anagramu porovnáva úroveň pozície. To môžeme dosiahnuť overením pozičného znaku prvého reťazca s každým pozičným znakom v druhom reťazci. Ak prvý reťazec obsahuje podobnú zhodu s druhým reťazcom, je označený za anagram. Poďme sa pozrieť na nasledujúci príklad.

def check_for_anagram(string1, string2):
    checking = True

    if len(string1) != len(string2):  
        checking = False  
        list1 = list(string2)  
    pos_str1 = 0
    while pos_str1 < len(string1) and checking: 
        pos_str2 = 0  
        found = False  
    while pos_str2 < len(list1) and not found:  
        if string1[pos_str1] == list1[pos_str2]:  
            found = True  
        else:  
            pos_str2 = pos_str2 + 1  
    if found:  
        list1[pos_str2] = None
    else:  
        checking = False
        pos_str1 = pos_str1 + 1  
    return checking  
  
  
string1 = "heart"
string2 = "earth"
print("String-1 : " , string1)  
print("String-2 : " , string2)  
result = check_for_anagram('heart','earth')  
if result:
    print("Both words are Anagram.")  
else:  
    print("Both words are not Anagram.")
# Output
String-1 :  heart
String-2 :  earth

Poďme si vysvetliť poslednú techniku na vyhľadanie anagramu dvoch reťazcov. Vo vnorenom while cykle vložíme do cyklov reťazec na overovací proces.

Vonkajší while cyklus je použitý na spracovanie jedného z reťazcov a vnútorný cyklus je použítý pre druhý reťazec. Znak jedného reťazca sa porovná s každým znakom ďalšieho reťazca a tento proces pokračuje pri každom písmene prvého reťazca. Ak sú všetky znaky prvého reťazca zhodné so znakmi druhého reťazca, potom môžme reťazce označiť za anagramy

Táto technika je veľmi stabilný proces, pretože funguje na algoritmickej technike, ktorá presne posudzuje reťazce.
