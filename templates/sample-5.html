Python Matrix

Skúmanie matíc(matrices) v Pythone

V tomto článku si predstavíme Python Matrix. Spoločne sa pozrieme na operácie s maticami(matrices) implementované pomocou Python kódu.

Matica(Matrix) je obdĺžnikové 2-rozmerné pole, ktoré ukladá údaje do riadkov a stĺpcov. Do matice je možné uložiť ľubovoľné dátové typy, ako sú čísla, reťazce, výrazy atď.

Python neobsahuje vstavaný typ pre matice, takže matice vytvoríme pomocou viacerých listov.

V tejto ukáźke sa pozrieme na nasledujúce operácie s maticami:

  • Sčítanie matíc
  • Odčítanie matíc
  • Násobenie matíc
  • Transpozícia matíc
  • Tvorba matíc pomocou Numpy knižnice

Nasledujúca matica má rozmer 2x2, čo znamená, že má dva riadky a dva stĺpce:

[[1,2],  
 [3,4]]

Matrix(maticu) v Pythone môžeme vytvoriť pomocou vnoreného listu. Všetky prvky sú uzavreté v hranatých zátvorkách ([]) a oddelené čiarkou. Pozrime sa na nasledujúce príklady: 

matrix = [["cat", 47, 21, 98],  
          ["dog", 541, 81.70, 27],  
          [365, "turtle", 58, 12]]

Vytvorili sme maticu v rozmere 3x3 pomocou vnoreného listu. Ako vidíme, naša matica pozostáva z čísel a reťazcov.

V nasledujúcom príklade vypíšeme vo výstupe každý riadok zadefinovanej matice:

matrix = [["cat", 47, 21, 98],  
          ["dog", 541, 81.70, 27],  
          [365, "turtle", 58, 12]]
          
          print("The matrix is: ", matrix)
The matrix is:  [['cat', 47, 21, 98], ['dog', 541, 81.7, 27], [365, 'turtle', 58, 12]]

V nasledujúcom príklade vypíšeme vo výstupe posledný prvok každého riadku:

matrix = [["cat", 47, 21, 98],  
          ["dog", 541, 81.70, 27],  
          [365, "turtle", 58, 12]]

matrix_length = len(matrix)  
          
#To read the last element from each row. 
for i in range(matrix_length):  
    print(matrix[i][-1])
98
27
12

Vyššie uvedenou metódou pomocou for cyklu a vstavanej funkcie len() môžeme vypísať ľubovoľný riadok, stĺpec alebo konkrétne prvky zadefinovanej matice.

V nasledujúcom príklade sčítame dve matice pomocou vnoreného for cyklu:

matrix1 = [[74, 15, 69],   
           [69, 8, 4],   
           [98, 14, 75]]  
  
matrix2 = [[69, 44, -8],  
           [3, 47, -9],   
           [-3, 7 , 88]]  
  
matrix3  = [[0,0,0],  
            [0,0,0],  
            [0,0,0]]    
  
#To Add mat1 and mat2 matrices  
for i in range(len(matrix1)):  
    for k in range(len(matrix2)):  
        matrix3[i][k] = matrix1[i][k] + matrix2[i][k]  
  
#To Print the matrix  
print("The sum of matrix1 and matrix2 =", matrix3)
The sum of matrix1 and matrix2 = [[143, 59, 61], [72, 55, -5], [95, 21, 163]]

V ukážke vyššie sme si zadefinovali dve matice v rozmere 3X3 a následne inicializovali tretiu 3X3 maticu, v ktorej uložíme výsledok sčítania prvých dvoch matíc. Na iteráciu matíc sme použili vnorený for cyklus. V prvej iterácii sa sčíta prvý prvok matice matrix1 a prvý prvok matice matrix2 a vnorená iterácia bude pokračovať, až kým sa nesčítajú všetky prvky oboch matíc.

V nasledujúcom príklade tentokrát dve matice vynásobíme. Princíp je rovnaký ako v predchádzajúcom príklade, len s rozdielom, že zmeníme operátor sčítania + na operátor násobenia *: 

matrix1 = [[74, 15, 69],   
           [69, 8, 4],   
           [98, 14, 75]]  
  
matrix2 = [[69, 44, -8],  
           [3, 47, -9],   
           [-3, 7 , 88]]  
  
matrix3  = [[0,0,0],  
            [0,0,0],  
            [0,0,0]] 
  
#To Add mat1 and mat2 matrices  
for i in range(len(matrix1)):  
    for k in range(len(matrix2)):  
        matrix3[i][k] = matrix1[i][k] * matrix2[i][k]  
  
#To Print the matrix  
print("The sum of matrix1 and matrix2 =", matrix3)
The sum of matrix1 and matrix2 = [[5106, 660, -552], [207, 376, -36], [-294, 98, 6600]]

V nasledujúcom príklade sa pozrieme na transpozíciu riadkov a stĺpcov matice. Transpozícia je operácia, pri ktorej sa riadok danej matice prevedie na stĺpec matice a naopak: 

matrix = [[8, 9],  
          [2, 9],  
          [1, 6]] 
  
result = [[0,0,0],  
          [0,0,0]] 
  
# iterate through rows  
for i in range(len(matrix)):  
   # iterate through columns  
   for j in range(len(matrix[0])):  
       result[j][i] = matrix[i][j]  
  
for x in result:  
   print(x)
[8, 2, 1]
[9, 9, 6]

V nasledujúcom príklade sa pozrieme na transpozíciu riadkov a stĺpcov matice pomocou generátorovej notácie (List Comprehension) a to použitím iba jedného riadku kódu: 

matrix = [[8, 9],  
          [2, 9],  
          [1, 6]]
  
result = [[matrix[j][i] for j in range(len(matrix))] for i in range(len(matrix[0]))]  
  
for x in result:  
   print(x)
[8, 2, 1]
[9, 9, 6]

Výstup je rovnaký ako v predošlom príklade. Použitím generátorovej notácie (List Comprehension) sme dosiahli zníženie počtu riadkov kódu na jeden a transponovali maticu.

Následne sa vo vnorenom for cykle prvky vkladajú do prázdnej matice pomocou metódy append(). Druhý vnorený for cyklus je použítý na výpis zadaných hodnôt vo forme matice.

Tvorba matíc pomocou knižnice Numpy

Na úvod jednoduchý príklad matice vytvorenej pomocou Numpy knižnice:

import numpy as np

matrix = np.array([[26, -15, 7], [78, -9, 21], [3, 13, 4]])

print("The matrix is: ")  
print(matrix)
The matrix is: 
[[ 26 -15   7]
 [ 78  -9  21]
 [  3  13   4]]

Všetky operácie s maticami, ako je sčítanie, odčítanie, transpozícia, segmentácia atď., môžeme vykonávať pomocou metódy numpy.array().

Poďme sa pozrieť na nasledujúci príklad. Vytvoríme dve matice pomocou funkcie numpy.array() a sčítame ich pomocou operátora sčítania +: 

import numpy as np

matrix1 = np.array([[4, 7, 9], [3, -2, 21], [15, 22, 18]])  

matrix2 = np.array([[3, -19, 17], [11, 55, 33], [23, -31, 42]])  

matrix3 = matrix1 + matrix2    

print("The matrix addition is: ")  
print(matrix3)
The matrix addition is: 
[[  7 -12  26]
 [ 14  53  54]
 [ 38  -9  60]]

Poďme sa pozrieť na nasledujúci príklad. Na znásobenie dvoch matíc použijeme metódu numpy.dot(). Jedná sa o bodové násobenie matíc matrix1 a matrix2: 

import numpy as np

matrix1 = np.array([[2, 3], [3, 5]])

matrix2 =  np.array([[6, -2], [22, 44]]) 

matrix3 = matrix1.dot(matrix2)   

print("The matrix is:")  
print(matrix3)
The matrix is:
[[ 78 128]
 [128 214]]

Teraz sa pozrieme na segmentáciu matíc a to dosiahneme rovnakým spôsobom ako to robíme v štandardnom Python liste. Segmentácia znamená načítanie prvkov od nejakého indexu po ďalší index. Stačí zadať začiatočný a konečný bod indexu nasledovne: [start: end]. V príkladne nižšie použijeme jednoduché pole na bližšie pochopenie: 

import numpy as np  
  
arr = np.array([15,30,25,74,32,17,99])

print(arr[1:4]) # It will print the elements from 1 to 4  
print(arr[:5]) # It will print the elements from 0 to 4  
print(arr[2:]) # It will print the elements from 2 to length of the array.
[30 25 74]
[15 30 25 74 32]
[25 74 32 17 99]

Teraz implementujeme segmentáciu na maticu. Docielime to tak, že zadáme index segmentácia pre riadky aj stĺpce matice.

Pozrime sa na nasledujúci príklad segmentácie matice:

import numpy as np

matrix = np.array([[ 6, 32, 58, 17, 34],  
                   [35, 16, 85,-11, 54],  
                   [23, 25, 42, 24,-20],  
                   [ 7,-12, 36, 17, 55]])

print(matrix[1:3, 2:4])
[[ 85 -11]
 [ 42  24]]

Vo výstupe vyššie sme vypísali tretí a štvrtý prvok druhého a tretieho riadku.
