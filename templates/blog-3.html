Čo je knižnica Pandas?

Úvod do Pandas

Pandas (čo je skratka pre „panel data"“) je jednou z najdôležitejších knižníc, ktorú by sme mali mať v malíčku v prípade, ak sme sa rozhodli učiť programovací jazyk Python.

Knižnica je známa pre veľmi užitočnú dátovú štruktúru, ktorá sa nazýva Pandas Dátový rámec. Pandas tiež umožňuje Python vývojárom ľahko narábať s tabuľkovými dátami v Python skripte.

Tento článok vás naučí základy Pandas knižnice, ktorú dnes môžeme využiť na vytváranie dátovo orientovaných Python aplikácií.

Pandas Série

V tejto časti sa budeme venovať Pandas Sériám, ktoré sú základnou súčasťou Pandas knižnice pre programovanie v jazyku Python.

Čo sú Pandas Série?

Pandas Série sú špeciálnym typom dátovej štruktúry dostupnej v Pandas knižnici. Pandas Série sú podobné NumPy poliam až na to, že im môžeme zadať index názvu alebo index času a nielen číselný index.

Importy, ktoré budeme potrebovať pri práci s Pandas Sériami

Aby sme mohli pracovať s Pandas Sériami, je potrebné importovať NumPy a Pandas knižnice, a to nasledovne:

import numpy as np
import pandas as pd

Po zbytok tohoto článku budeme predpokladať, že obidve tieto knižnice boli importované pred spustením akýchkoľvek nasledujúcich blokov kódu.

Ako vytvoriť Pandas Série

Existuje množstvo rôznych spôsobov, ako vytvoriť Pandas Série. Na všetky z nich sa pozrieme v tejto časti.

Najprv si vytvoríme niekoľko štartovacích premenných - vytvoríme dva listy, NumPy pole a slovník.

labels = ['a', 'b', 'c']
my_list = [10, 20, 30]
arr = np.array([10, 20, 30])
d = {'a':10, 'b':20, 'c':30}

Najjednoduchší spôsob, ako vytvoriť Pandas Série, je použitím metódy pd.Series() v ktorej zadáme Python list ako argument. To vykonáme pomocou nižšie uvedenej premennej my_list: 

pd.Series(my_list)

Ak to spustíte vo svojom Jupyter notebooku, zistíte, že výstup je mierne odlišný od výstupu pre normálny Python list: 

0    10
1    20
2    30
dtype: int64

Výstup zobrazený vyššie je jasne navrhnutý tak, aby sa zobrazoval ako dva stĺpce. Druhým stĺpcom sú dáta z premennej my_list. Čo myslíte, čo predstavuje prvý stĺpec?

Jednou z kľúčových výhod používania Pandas Sérií oproti NumPy poliam je, že umožňujú označovanie. Ako ste mohli uhádnuť, prvý stĺpec je stĺpec označení.

Označenia môžeme pridať k Pandas Sériám pomocou argumentu index, nasledovne:

pd.Series(my_list, index=labels)
#Pamätajte - 'labels' list sme už vytvorili vyššie v tejto sekcii

Výstup tohto kódu je uvedený nižšie:

a    10
b    20
c    30
dtype: int64

Spýtajme sa teraz prečo by sme chceli používať označenia v Pandas Sériách? Hlavnou výhodou je, že vám umožňia odkazovať na prvok zo série pomocou jeho označení namiesto jeho číselného indexu. Aby bolo jasné, po zadaní označení v Pandas Sériách môžeme použiť buď ich číselný index, alebo ich označenie.

Príklad je uvedený nižšie.

Series = pd.Series(my_list, index=labels)
Series[0]
#Výstup je 10
Series['a']
#Výstup je taktiež 10

Možno ste si všimli, že schopnosť odkazovať na prvok série pomocou jeho označenia je podobná tomu, ako sa odkazuje na hodnoty pri Python slovníkoch. Kvôli tejto podobnosti ich fungovania môžeme taktiež zadať Python slovník ako argument na vytvorenie Pandas Sérií. Ako príklad použijeme slovník d = {'a': 10, 'b': 20, 'c': 30}, ktorý sme si už vytvorili vyššie: 

pd.Series(d)

Výstup tohto kódu je:

a    10
b    20
c    30
dtype: int64

Zatiaľ vám asi ešte nieje jasné, prečo sme sa zaoberali dvomi novými dátovymi štruktúrami (NumPy polia a Pandas Série), ktoré sú si veľmi podobné. Preto sa v nasledujúcej časti pozrieme na hlavné výhody Pandas Sérií oproti NumPy poliam.

Hlavná výhoda Pandas Sérií oproti NumPy poliam

Aj keď sme sa s tým zatiaľ nestretli, NumPy polia sú veľmi limitované jednou charakteristikou: každý prvok NumPy poľa musí byť rovnakého typu dátovej štruktúry. Inak povedané , všetky prvky NumPy poľa musia byť buď stringy, celé čísla alebo boolean hodnoty.

Pandas Série týmto obmedzením netrpia, pretože sú v skutočnosti vysoko flexibilné.

Napríklad môžeme do Pandas Sérií zadať tri vstavané Python funkcie bez toho, aby sme dostali chybu:

pd.Series([sum, print, len])

Tu je výstup tohto kódu:

0      <built-in function sum>
1      <built-in function print>
2      <built-in function len>
dtype: object

Je zrejmé, že vyššie uvedený príklad je veľmi nepraktický a nejedná sa o niečo, čo by sme v praxi niekedy vykonali. Je to však vynikajúci príklad flexibility dátovej štruktúry Pandas Sérií.

Pandas Dátové rámce

NumPy umožňuje vývojárom pracovať s jednorozmernými NumPy poliami (nazývané vektory) a s dvoj a viacrozmernými NumPy poliami (nazývané matice). V poslednej časti sme skúmali Pandas Série, ktoré sú podobné jednorozmerným NumPy poliam.

V tejto časti sa pozrieme na Pandas Dátové rámce, ktoré sú podobné dvojrozmerným NumPy poliam - ale s oveľa väčšou funkcionalitou. Dátové rámce sú najdôležitejšou dátovou štruktúrou v Pandas knižnici, takže celej tejto časti venujte zvýšenú pozornosť.

Čo je Pandas Dátový rámec?

Pandas Dátový rámec je dvojrozmerná dátová štruktúra, ktorá má označenia pre svoje riadky aj stĺpce. Pre tých, ktorí poznajú Microsoft Excel, Google Sheets alebo iný tabuľkový softvér, sú Dátové rámce veľmi podobné.

Tu môžeme vidieť príklad Pandas Dátového rámca, ktorý sa zobrazuje v Jupyter notebooku.

Teraz prejdeme procesom vytvorenia tohto Dátového rámca krok za krokom.

Najskôr budeme musieť importovať knižnice NumPy a Pandas. Urobili sme to už predtým, ale v prípade, že si nie ste istí, tu je ďalší príklad, ako to urobiť: 

import numpy as np
import pandas as pd

Budeme tiež musieť vytvoriť zoznamy pre názvy riadkov a stĺpcov. Môžeme to urobiť pomocou Python listov: 

rows = ['X','Y','Z']
cols = ['A', 'B', 'C', 'D', 'E']

Ďalej budeme musieť vytvoriť NumPy pole, ktoré obsahuje údaje obsiahnuté v bunkách Dátového rámca. Použijeme na to NumPy metódu np.random.randn(). Túto metódu sme zabalili aj do metódy np.round() (kde sme zadali 2 ako druhý argument), ktorá zaokrúhľuje každý dátový bod na dve desatinné miesta a umožňuje oveľa ľahšiu čitateľnosť dátovej štruktúry.

Toto je finálna funkcia, ktorá vygeneruje dáta.

data = np.round(np.random.randn(3,5),2)

Keď toto máme hotové všetky potrebné premenné môžeme zabaliť do metódy pd.DataFrame() a vytvoriť tak náš prvý Dátový rámec! 

pd.DataFrame(data, rows, cols)

Stále nám však ostáva veľa vecí, ktoré treba prebrať, poďme sa teda pozrieť na tento príklad trochu podrobnejšie.

Za prvé, nie je potrebné vytvárať každú premennú mimo samotného Dátového rámca. Tento Dátový rámec sme mohli vytvoriť v jednom riadku nasledovne: 

pd.DataFrame(np.round(np.random.randn(3,5),2), ['X','Y','Z'], ['A', 'B', 'C', 'D', 'E'])

Z uvedeného vyplýva, že deklarovanie každej premennej osobitne uľahčuje čítateľnosť kódu.

Za druhé, možno by vás zaujímalo, či je potrebné vložiť riadky do metódy pd.DataFrame() pred stĺpcami. Je to skutočne nevyhnutné. Ak by ste sa pokúsili spustiť kód pd.DataFrame(data, cols, rows), váš Jupyter notebook vygeneruje nasledujúcu chybu: 

ValueError: Shape of passed values is (3, 5), indices imply (5, 3)

Ďalej sa pozrieme na vzťah Pandas Sérií a Pandas Dátových rámcov.

Vzťah Pandas Sérií a Pandas Dátových rámcov

Pozrime sa ešte raz na Pandas Dátový rámec, ktorý sme práve vytvorili:

Ak by sme mali slovne opísať Pandas Dátový rámec, jeden spôsob by bol “sada označených stĺpcov obsahujúcich dáta, kde každý stĺpec zdieľa rovnaký index riadkov“.

Je zaujímavé, že každý z týchto stĺpcov je vlastne Pandas Séria! Môžeme teda upraviť našu definíciu Pandas Dátového rámca tak, aby zodpovedala jeho formálnej definícii:

"Sada Pandas Sérií, ktorá zdieľa rovnaký index."

Index a priraďovanie v Pandas Dátových rámcoch

Načítať konkrétnu sériu z Pandas Dátového rámca môžeme pomocou hranatých zátvoriek, rovnako ako načítavame prvok z Python listu. Niekoľko príkladov je uvedených nižšie: 

df = pd.DataFrame(data, rows, cols)
df['A']
"""
Výstup:
X   -0.66
Y   -0.08
Z    0.64
Name: A, dtype: float64
"""
df['E']
"""
Výstup:
X   -1.46
Y    1.71
Z   -0.20
Name: E, dtype: float64
"""

Čo keby sme chceli načítať viac stĺpcov z Pandas Dátového rámca? Môžete vložiť list stĺpcov buď priamo do hranatých zátvoriek - napríklad df[['A', 'E']] - alebo deklarovať premennú mimo hranatých zátvoriek nasledovne: 

columnsIWant = ['A', 'E']
df[columnsIWant]

#Výstup je Dátový rámec, obsahujúci stĺpce A a E

Môžeme tiež načítať konkrétny prvok konkrétneho riadku pomocou zložených zátvoriek. Napríklad, ak chceme dostať prvok z riadku A na indexe X (čo je prvok v ľavej hornej bunke Dátového rámca), môžeme sa k nemu dostať nasledovne. df['A']['X'].

Niekoľko ďalších príkladov je uvedených nižšie.

df['B']['Z']
#Výstup 1.34
df['D']['Y']
#Výstup -0.64

Ako vytvoriť a odstrániť stĺpce v Pandas Dátovom rámci

Nový stĺpec môžeme vytvoriť v Pandas Dátovom rámci deklarovaním nového stĺpca a k nemu následne priradíme nové Pandas Série.

Napríklad v nasledujúcom bloku kódu vytvoríme nový stĺpec s názvom „A + B“, ktorý je súčtom stĺpcov A a B: 

df['A + B'] = df['A'] + df['B']
df 

#Posledný riadok vypíše nový Dátový rámec

Tu je výstup z tohto bloku kódu:

Ak chceme tento stĺpec z Pandas Dátového rámca odstrániť, použijeme metódu pd.DataFrame.drop().

Poznámka: táto metóda je predvolená na odstránenie riadkov, nie stĺpcov. Ak chceme zmeniť nastavenie metódy tak, aby fungovala na stĺpcoch, musíte pri metóde zadať argument axis = 1. 

df.drop('A + B', axis = 1)

Je veľmi dôležité poznamenať, že táto metóda odstránenia vlastne nemení samotný Dátový rámec. Dôkazom toho je, výstup premennej df kde vidíme, že stále obsahuje stĺpec A + B: 

df

Dôvodom, prečo drop() (a mnoho ďalších metód Dátových rámcov!) predvolene nemení štruktúru dát, je zabránenie náhodnému odstráneniu dát.

Existujú dva spôsoby, ako dosiahnuť, aby Pandas automaticky prepísala aktuálny Dátový rámec.

Prvým je zadanie argumentu inplace = True, nasledovne:

df.drop('A + B', axis=1, inplace=True)

Druhým je použitie operátora priradenia, ktorý manuálne prepíše existujúcu premennú, nasledovne:

df = df.drop('A + B', axis=1)

Obe možnosti sú správne, ale druhú možnosť používam častejšie, pretože je ľahšie zapamätateľná.

Metódu drop() je možné použiť aj na odstránenie riadkov. Napríklad riadok Z môžeme odstrániť nasledovne: 

df.drop('Z')

Ako načítať riadok z Pandas Dátového rámca

Už sme videli, že sa môžeme dostať k špecifickému stĺpcu Pandas Dátového rámca pomocou hranatých zátvoriek. Teraz sa pozrieme na to, ako sa dostať ku konkrétnemu riadku Pandas Dátového rámca, s podobným cieľom vygenerovať Pandas Série z väčšej dátovej štruktúry.

K riadkom Dátového rámca sa môžeme dostať cez označenie riadkov pomocou atribútu loc spolu s hranatými zátvorkami. Príklad je uvedený nižšie. 

df.loc['X']

Tu je výstup tohto kódu:

A   -0.66
B   -1.43
C   -0.88
D    1.60
E   -1.46
Name: X, dtype: float64

K riadkom Dátového rámca sa taktiež môžeme dostať cez číselný index pomocou atribútu iloc spolu s hranatými zátvorkami. Príklad je uvedený nižšie. 

df.iloc[0]

Ako iste predpokladáte, tento kód má rovnaký výstup ako náš predchádzajúci príklad:

A   -0.66
B   -1.43
C   -0.88
D    1.60
E   -1.46
Name: X, dtype: float64

Ako určiť počet riadkov a stĺpcov v Pandas Dátovom rámci

V mnohých prípadoch je potrebné poznať tvar Pandas Dátového rámca. Tým mám na mysli počet stĺpcov a riadkov v dátovej štruktúre.

Pandas má vstavaný atribút zvaný shape, ktorý nám umožňuje určiť tvar nášho Dátového rámca: 

df.shape
#Výstup (3, 5)

Segmentácia Pandas Dátových rámcov

Už sme sa naučili, ako načítať riadky, stĺpce a prvky z Pandas Dátového rámca. V tejto časti sa pozrieme na to, ako načítať podmnožinu Dátového rámca. Konkrétne vyberieme prvky zo stĺpcov A a B a riadkov X a Y.

Najprv vyberieme stĺpce A a B:

df[['A', 'B']]

A následne k tomu vyberieme riadky X a Y:

df[['A', 'B']].loc[['X', 'Y']]

A sme hotoví!

Podmienený výber pomocou Pandas Dátového rámca

Ak si spomínate na našu diskusiu o NumPy poliach, boli sme schopní načítať určité prvky poľa pomocou podmienených operátorov. Napríklad, ak sme mali pole NumPy nazvané arr a chceli sme načítať iba hodnoty poľa, ktoré boli väčšie ako 4, mohli sme použiť príkaz arr[arr > 4].

Pandas Dátové rámce sa obsahujú podobnú syntax. Napríklad, ak by sme chceli načítať hodnoty, ktoré sú väčšie ako 0,5, mohli by sme zadať príkaz df > 0,5, aby sme získali nasledujúci výstup:

Môžeme tiež vygenerovať nový Pandas Dátový rámec, ktorý obsahuje normálne hodnoty, ktoré sú určené ako True, a NaN - čo znamená Not a Number - hodnoty, ktoré sú určené ako False. Spravíme to tak, že podmienku zadáme do Dátového rámca pomocou hranatých zátvoriek nasledovne: 

df[df > 0.5]

Tu je výstup tohto kódu:

Podmienený výber môžeme tiež použiť na načítanie podmnožiny Dátového rámca, kde je v zadanom stĺpci splnená konkrétna podmienka.

Aby som bol konkrétnejší, povedzme, že sme chceli podmnožinu Dátového rámca, kde hodnota v stĺpci C je menšia ako 1 ale s platnosťou iba pre riadok X.

Môžeme získať pole boolean hodnôt v rámci tejto podmienky nasledovne:

df['C'] < 1

Tu je výstup:

X     True
Y    False
Z    False
Name: C, dtype: bool

Normálne hodnoty Dátového rámca vo vzťahu k predchádzajúcej podmienke môžeme získať aj zadaním príkazu df[df['C'] < 1], ktorý vygeneruje iba prvý riadok Dátového rámca (pretože toto je jediný riadok určený ako True pre stĺpec C:

Pri použití podmieneného výberu môžeme tiež spojiť dohromady viac podmienok. V rámci Pandas to robíme pomocou operátora &. Nemôžeme použiť normálny and operátor Pythonu, pretože v tomto prípade neporovnávame dve boolean hodnoty. Namiesto toho porovnávame dve Pandas Série, ktoré obsahujú boolean hodnoty, a preto sa namiesto operátora and používa znak &.

Ako príklad viacnásobného podmieneného výberu môžeme načítať podmnožinu Dátového rámca, ktorá spĺňa podmienky df['C'] > 0 a df['A'] > 0 nasledovne: 

df[(df['C'] > 0) & (df['A']> 0)]

Ako modifikovať index Pandas Dátového rámca

Existuje niekoľko spôsobov, ako môžeme modifikovať index Pandas Dátového rámca.

Najzákladnejšie je vynulovať index na jeho predvolené číselné hodnoty. To spravíme pomocou metódy reset_index(): 

df.reset_index()

Treba si uvedomiť, že táto metóda vytvorí nový stĺpec v Dátovom rámci s názvom index, ktorý obsahuje predchádzajúce označenia indexu:

Všimnite si, že reset_index(): nezmení pôvodný Dátový rámec tak ako ďalšie operácie Dátového rámca, ktoré sme skúmali, pokiaľ (1) si nevynútime použitie = priraďovacieho operátora alebo (2) pokiaľ nepridáme atribút inplace = True.

Taktiež môžeme nastaviť už existujúci stĺpec ako index Dátového rámca pomocou metódy set_index(). Stĺpec A ako index môžeme nastaviť pomocou nasledujúceho kódu: 

df.set_index('A')

Tu treba mať na pamäti tri veci:

  • set_index(): nezmení pôvodný Dátový rámec, pokiaľ (1) si nevynútime použitie = priraďovacieho operátora alebo (2) nepridáme atribút inplace = True.
  • Pokiaľ nespustíme najskôr reset_index() metódu, vykonaním operácie set_index() s inplace = True alebo vynúteným = priraďovacím operátorom trvale prepíšeme naše aktuálne hodnoty indexu.
  • Ak chceme svoj index premenovať na označenia, ktoré momentálne nie sú obsiahnuté v stĺpci, môžeme to urobiť tak, že (1) vytvoríme NumPy pole s týmito hodnotami, (2) pridáme tieto hodnoty ako nový riadok Pandas Dátového rámca a (3) spustíme set_index() operáciu.

Pandas metóda dropna()

Knižnica Pandas obsahuje vstavanú metódu dropna(). Pri jej použití, metóda odstráni všetky riadky Dátového rámca, ktoré obsahujú hodnotu NaN.

Ako príklad použijeme metódu dropna() na náš df Dátový rámec:

df.dropna()

Všimnite si, že rovnako ako ostatné operácie Dátového rámca, ktoré sme skúmali, dropna() nemení pôvodný Dátový rámec, pokiaľ (1) si nevynútime použitie = priraďovacieho operátora alebo (2) pokiaľ nepridáme atribút inplace = True.

Môžeme tiež odstrániť aj všetky stĺpce, ktoré majú chýbajúce hodnoty a to zadaním argumentu axis = 1 pri metóde dropna() nasledovne: 

df.dropna(axis=1)

Pandas metóda fillna()

V mnohých prípadoch budeme chcieť namiesto odstránenia chýbajúcich hodnôt v Pandas Dátovom rámci tieto chýbajúce hodnoty nahradiť. Na to slúži metóda fillna().

Ako príklad vyplňme každú chýbajúcu hodnotu v našom Dátovom rámci znakom ?:

df.fillna('?')

Je zrejmé, že v zásade neexistuje situácia, kde by sme chceli nahradiť chýbajúce údaje s emodži. Bol to iba zábavný príklad.

Namiesto toho vo väčšine prípadov nahradíme chýbajúcu hodnotu buď:

  • Priemernou hodnotou celého Dátového rámca
  • Priemernou hodnotou daného riadku Dátového rámca

Obidve metódy si ukážeme nižšie.

Ak chceme vyplniť chýbajúce hodnoty priemernou hodnotou celého Dátového rámca, použijeme nasledujúci kód: 

df.fillna(df.mean())

Ak chceme vyplniť chýbajúce hodnoty v konkrétnom stĺpci priemernou hodnotou tohto stĺpca, použijeme nasledujúci kód (kód je pre stĺpec A): 

df['A'].fillna(df['A'].mean())

Pandas groupby metóda

V tejto časti budeme diskutovať o tom, ako použíť Pandas groupby() metódu.

Pandas prichádza so vstavanou funkciou groupby(), ktorá nám umožňuje zoskupovať riadky na základe stĺpcov a vykonávať na nich agregačné funkcie. Môžeme napríklad vypočítať sumu všetkých riadkov, ktoré majú v stĺpci ID hodnotu 1.

Pre každého, kto je oboznámený s jazykom SQL na dotazovanie databáz, je metóda Pandas groupby() veľmi podobná SQL príkazu GROUP BY.

Metódu Pandas groupby() najjednoduchšie pochopíme pomocou príkladu. Budeme používať nasledujúci Dátový rámec: 

df = pd.DataFrame([ ['Google', 'Sam', 200],
                    ['Google', 'Charlie', 120],
                    ['Salesforce','Ralph', 125],
                    ['Salesforce','Emily', 250],
                    ['Adobe','Rosalynn', 150],
                    ['Adobe','Chelsea', 500]])

df.columns = ['Organization', 'Salesperson Name', 'Sales']

df

Tento Dátový rámec obsahuje informácie o predaji pre tri samostatné organizácie: Google, Salesforce a Adobe. Na získanie súhrnných dát o predaji pre každú konkrétnu organizáciu použijeme metódu groupby().

Na začiatok budeme musieť vytvoriť groupby objekt, čo je dátová štruktúra, ktorá Pythonu hovorí, do ktorého stĺpca chceme zoskupiť Dátový rámec. V našom prípade je to stĺpec Organizácia, takže vytvoríme groupby objekt nasledovne: 

df.groupby('Organization')

Ak uvidíme výstup, ktorý vyzerá nasledovne, budeme vedieť, že sme objekt vytvorili úspešne:

<pandas.core.groupby.generic.DataFrameGroupBy object at 0x113f4ecd0>

Po vytvorení groupby objektu môžeme s týmto objektom vykonávať operácie a vytvoriť Dátový rámec so súhrnnými informáciami o skupinách organizácií. Niekoľko príkladov je uvedených nižšie: 

df.groupby('Organization').mean()

#Priemer (alebo stredná hodnota) sales stĺpca

df.groupby('Organization').sum()

#Suma sales stĺpca

df.groupby('Organization').std()

#Štandardná odchýlka sales stĺpca

Pretože všetky vyššie uvedené operácie sú číselné, stĺpec Salesperson Name bude automaticky ignorovaný, pretože obsahuje iba stringy.

Tu je niekoľko ďalších agregačných funkcií, ktoré dobre fungujú s Pandas groupby() metódou: 

df.groupby('Organization').count()

#Spočíta počet záznamov

df.groupby('Organization').max()

#Výstup maximálnej hodnoty

df.groupby('Organization').min()

#Výstup minimálnej hodnoty

Pandas merge metóda

V tejto časti sa dozvieme, ako zlúčiť Pandas Dátové rámce.

Dátové rámce, ktoré použijeme v tejto časti

V tejto časti použíjeme nasledujúce dva Pandas Dátové rámce:

import pandas as pd

leftDataFrame = pd.DataFrame({'key': ['K0', 'K1', 'K2', 'K3'],
                                'A': ['A0', 'A1', 'A2', 'A3'],
                                'B': ['B0', 'B1', 'B2', 'B3']})

rightDataFrame = pd.DataFrame({'key': ['K0', 'K1', 'K2', 'K3'],
                                 'C': ['C0', 'C1', 'C2', 'C3'],
                                 'D': ['D0', 'D1', 'D2', 'D3']})

Stĺpce A, B, C a D obsahujú skutočné údaje, zatiaľ čo key stĺpec je spoločný pre oba Dátové rámce. Zlúčiť dva Dátové rámce znamená prepojiť ich pozdĺž toho stĺpca, ktorý je pre rámce spoločný.

Ako zlučovať Pandas Dátové rámce

Zlúčiť dva Dátové rámce môžeme pozdĺž ich spoločného stĺpca. Pre každého, kto je oboznámený s jazykom SQL, je metóda Pandas merge() veľmi podobná SQL príkazu INNER JOIN.

Ak nepoznáte jazyk SQL, nerobte si starosti, pretože syntax zlúčenia je v skutočnosti veľmi jednoduchá. Vyzerá nasledovne: 

pd.merge(leftDataFrame, rightDataFrame, how='inner', on='key')

Poďme si rozdeliť štyri argumenty, ktoré sme použili v metóde merge():

  1. leftDataFrame: Toto je Dátový rámec, ktorý by sme chceli zlúčiť na ľavú stranu.
  2. rightDataFrame: Toto je Dátový rámec, ktorý by sme chceli zlúčiť na pravú stranu.
  3. how=inner: Toto je typ zlúčenia, ktorý operácia vykoná. Existuje niekoľko typov zlúčenia, ale v tomto článku sa budeme venovať iba inner zlúčeniu.
  4. on='key' Toto je stĺpec, pozdĺž ktorého chceme zlúčovať. Keďže key je jediný spoločný stĺpec medzi dvoma Dátovými rámcami, je to jediná možnosť, ktorá pripadá do úvahy na vykonanie zlúčenia.

Pandas join metóda

V tejto časti sa dozvieme, ako k sebe pripojiť Pandas Dátové rámce.

Dátové rámce, ktoré použijeme v tejto časti

leftDataFrame = pd.DataFrame({  'A': ['A0', 'A1', 'A2', 'A3'],

Na pripojenie týchto dvoch Dátových rámcov k sebe môžeme použiť nasledujúci kód:

Ako nájsť unikátne hodnoty v Pandas Sérii

Pandas obsahuje vynikajúcu metódu nazvanú unique(), ktorú je možné použiť na vyhľadanie unikátnych hodnôt v Pandas Sérii. Treba však poznamenať, že táto metóda funguje iba na Série, a nie na Dátové rámce. Ak by sme chceli použiť túto metódu na Dátový rámec, dostaneme chybu: 

df.unique()

#Výstup AttributeError: 'DataFrame' object has no attribute 'unique'

Keďže každý stĺpec Pandas Dátového rámca je vlastne Pandas Séria, môžeme na konkrétny stĺpec použiť metódu unique(),nasledovne: 

df['col2'].unique()

#Výstup array([2, 7, 3])

Pandas taktiež obsahuje samostatnú metódu nunique(), ktorá zráta počet unikátnych hodnôt v Sérii a vypíše túto hodnotu ako celé číslo. Napríklad: 

df['col2'].nunique()

#Výstup 3

Je zaujímavé, že metóda nunique() je úplne rovnaká ako len(unique()), ale vzhľadom na to, že je to dosť zaužívaná operácia, tak sa Pandas komunita rozhodla pre tento prípad vytvoriť špecifickú metódu.

Použitie Pandas metódy apply

Metóda apply() je jednou z najvýkonnejších metód dostupných v Pandas knižnici. Umožňuje nám aplikovať vlastnú funkciu na každý prvok Pandas Série.

def exponentify(x):

df['col2'].apply(exponentify)

df['col3'].apply(len)

2    3
3    3
Name: col3, dtype: int64

Ako importovať súbory .csv pomocou Pandas

Súbory .csv môžeme importovať do Pandas Dátového rámca pomocou metódy read_csv(), nasledovne: 

import pandas as pd

pd.read_csv('stock_prices.csv')

Tento nový Dátový rámec môžeme tiež priradiť k premennej, na ktorú sa bude neskôr odkazovať, pomocou klasického = priraďovacieho operátora: 

new_data_frame = pd.read_csv('stock_prices.csv')

V programovacej knižnici Pandas je obsiahnuté množstvo metód na čítanie súborov. Ak by sme chceli importovať dáta z externého dokumentu, je pravdepodobné, že Pandas má vstavanú metódu aj na tento účel.

Niekoľko príkladov rôznych metód čítania je uvedených nižšie:

pd.read_json()
pd.read_html()
pd.read_excel()

Niektoré z týchto metód preskúmame v tejto časti neskôr.

Ak by sme chceli importovať súbor .csv, ktorý sa nenachádza priamo v našom pracovnom adresári, musíme mierne upraviť syntax metódy read_csv().

Ak je súbor hlbšie v priečinku ako v ktorom teraz pracujeme, musíme v argumente metódy read_csv() určiť úplnú cestu k súboru. Napríklad ak sa súbor stock_prices.csv nachádza v priečinku s názvom new_folder, mohli by sme ho importovať nasledovne: 

new_data_frame = pd.read_csv('./new_folder/stock_prices.csv')

Ak nieste oboznámení s prácou s adresárovými štruktúrami, . na začiatku cesty k súboru indikuje aktuálny adresár. Podobne .. indikuje jeden adresár nad aktuálnym adresárom a ... indikuje dva adresáre nad aktuálnym adresárom.

Použitím tejto syntaxe (pomocou bodiek) importujeme a odkazujeme na súbory, ktoré sú nad naším aktuálnym pracovným adresárom. Ako príklad si môžeme otvoriť Jupyter Notebook vo vnútri priečinka new_folder a do nadradeného priečinka vložte súbor stock_prices.csv. S týmto súborovým rozmiestnením môžeme importovať súbor stock_prices.csv pomocou nasledujúceho príkazu: 

new_data_frame = pd.read_csv('../stock_prices.csv')

Treba si uvedomiť, že táto adresárová syntax je rovnaká pre všetky typy importu súborov, takže keď budeme neskôr v tomto článku skúmať rôzne spôsoby importu, nebudeme sa vracať k tomu, ako importovať súbory z rôznych adresárov.

new_data_frame.to_csv('my_new_csv.csv', index = False)

Metódy read_csv() a to_csv() nám uľahčujú import a export dát zo súborov .csv vo vysokej miere. Ďalej v tejto časti uvidíme, že pre každú metódu čítania súborov, ktorá nám umožňuje importovať dáta, zvyčajne existuje zodpovedajúca funkcia, ktorá nám umožňuje tieto dáta aj uložiť!

Ako importovať súbory .json pomocou Pandas

Ak nemáte skúsenosti s prácou s veľkými dátovými setmi, asi nieste oboznámení s typom súboru JSON.

JSON je skratka pre JavaScript Object Notation. Súbory JSON sú veľmi podobné Python slovníkom.

Súbory JSON sú jedny z najbežnejšie používaných dátových typov medzi softvérovými vývojármi, pretože s nimi možno manipulovať v zásade pomocou každého programovacieho jazyka.

Pandas obsahuje metódu nazvanú read_json(), ktorá umožňuje veľmi jednoduchý import súborov JSON ako Pandas Dátových rámcov. Príklad je uvedený nižšie. 

json_data_frame = pd.read_json('stock_prices.json')

Ďalej sa naučíme, ako exportovať súbory JSON.

Ako exportovať súbory .json pomocou Pandas

Ako už iste vieme, pre každú metódu čítania súborov všeobecne existuje aj metóda na uloženie súborov. To znamená, že Dátový rámec môžeme uložiť aj do JSON súboru pomocou metódy to_json(). 

df.to_json('my_new_json.json')

Ďalej sa naučíme, ako pracovať so súbormi programu Excel - ktoré majú príponu súboru .xlsx.

Ako importovať súbory .xlsx pomocou Pandas

Pandas obsahuje metódu nazvanú read_excel(), ktorá umožňuje veľmi jednoduchý import Excel dokumentov ako Pandas Dátových rámcov. 

new_data_frame = pd.read_excel('stock_prices.xlsx')

Na rozdiel od metód read_csv() a read_json(), ktoré sme skúmali vyššie, môže metóda read_excel() prijať druhý argument. Dôvod, prečo read_excel() prijíma viac argumentov, je ten, že Excel tabuľky môžu obsahovať viac hárkov. Druhý argument určuje, ktorý hárok chceme importovať, a nazýva sa sheet_name.

Napríklad, ak by náš súbor stock_prices.xlsx mal druhý hárok s názvom Sheet2, tento hárok by sme importovali ako Pandas Dátový rámec nasledovne: 

new_data_frame.to_excel('stock_prices.xlsx', sheet_name='Sheet2')

Ak nezadáme žiadnu hodnotu pre argument sheet_name, tak metóda read_excel() predvolene naimportuje prvý hárok Excel tabuľky.

Pri importe Excel dokumentov je veľmi dôležité poznamenať, že Pandas importuje iba dáta. Nemôže importovať ďalšie funkcie Excelu, ako je formátovanie, vzorce alebo makrá. Pokus o import dát z Excel dokumentu, ktorý obsahuje tieto funkcie, môže spôsobiť zlyhanie Pandas knižnice.

Ako exportovať súbory .xlsx pomocou Pandas

Export Excel súborov je veľmi podobný ako ich import s rozdielom, že namiesto metódy read_excel() použijeme metódu to_excel(). Nižšie je uvedený príklad s použitím nášho náhodne vygenerovaného Dátového rámca df: 

df.to_excel('my_new_excel_file.xlsx')

Rovnako ako read_excel(), aj to_excel() prijíma druhý argument s názvom sheet_name, ktorý umožňuje určiť názov hárka, ktorý chceme uložiť. V príklade nižšie sme hárok nového súboru .xlsx pomenovali My New Sheet! a do metódy to_excel() ho zadáme nasledovne: 

df.to_excel('my_new_excel_file.xlsx', sheet_name='My New Sheet!')

Ak hodnotu pre sheet_name nezadáme, tak hárok bude predvolene pomenovaný Sheet1 (rovnako ako pri vytváraní nového Excel dokumentu pomocou skutočnej aplikácie).

Vstup a výstup externých dát (I/O) v Pandas

V poslednej časti tohto článku sme sa naučili, ako importovať dáta z .csv, .json a .xlsx súborov, ktoré boli uložené v našom lokálnom počítači. Budeme pokračovať a ukážeme si, ako môžeme importovať súbory bez toho, aby sme ich najskôr uložili na lokálny počítač a to sa nazýva externý import.

Čo je externý import a prečo je užitočný?

Externý import znamená vloženie súboru do Python skriptu bez toho, aby sme ho uložili vo svojom počítači.

Na prvý pohľad nemusí byť jasné, prečo by sme chceli radšej použiť externý import. Môže to však byť veľmi užitočné.

Dôvod, prečo je externý import užitočný z definície hovorí, že Python skript bude naďalej fungovať, aj keď importovaný súbor nie je uložený v našom počítači. To znamená, že môžeme poslať svoj kód kolegom alebo priateľom a bude stále fungovať správne.

V ďalšej časti si ukážeme, ako vykonávať externé importy v Pandas pre .csv, .json a .xlsx. súbory

Ako importovať externé .csv súbory

Najskôr prejdime do GitHub úložiska tohto článku. Otvorme priečinok stock_prices. Klikneme na súbor stock_prices.csv a potom klikneme na tlačidlo Raw file, ako je uvedené nižšie.

Dostanete sa na novú stránku, ktorá obsahuje dáta zo súboru .csv v rámci stock_prices.csv.

Ak chceme importovať tento externý súbor do Python skriptu, musíme najskôr skopírovať jeho URL adresu.

URL adresa by mala vyzerať takto:

https://raw.githubusercontent.com/nickmccullum/advanced-python/master/stock_prices/stock_prices.csv

Túto URL adresu môžeme zadať v metóde read_csv() ako argument na import dátového setu do Pandas Dátového rámca bez toho, aby sme najskôr dátový set uložili na svoj počítač: 

pd.read_csv('https://raw.githubusercontent.com/nicholasmccullum/advanced-python/master/stock_prices/stock_prices.csv')

Ako importovať externé .json súbory

Externé .json súbory môžeme importovať podobným spôsobom ako .csv súbory.

Najskôr skopírujme nespracovanú URL adresu z GitHubu. Mala by vyzerať takto:

https://raw.githubusercontent.com/nicholasmccullum/advanced-python/master/stock_prices/stock_prices.json

Potom zadáme túto URL adresu do metódy read_json() nasledovne:

pd.read_json('https://raw.githubusercontent.com/nicholasmccullum/advanced-python/master/stock_prices/stock_prices.json')

Ako importovať externé .xlsx súbory

Externé .xlsx súbory môžeme importovať podobným spôsobom ako .csv a .json súbory. Treba si však uvedomiť, že v GitHub rozhraní budeme musieť kliknúť na trochu iné miesto. Konkrétne budeme musieť kliknúť pravým tlačidlom myši na položku „View Raw“ a zvoliť možnosť „Copy Link Address,“, ako je uvedené nižšie.

Nespracovaná URL adresa bude vyzerať takto:

https://github.com/nicholasmccullum/advanced-python/blob/master/stock_prices/stock_prices.xlsx?raw=true

Potom zadáme túto URL adresu do metódy read_excel() nasledovne:

d.read_excel('https://github.com/nicholasmccullum/advanced-python/blob/master/stock_prices/stock_prices.xlsx?raw=true')

Nevýhody externého importu

Externý import znamená, že nemusíme najskôr uložiť importovaný súbor na lokálny počítač, čo je nepochybná výhoda.

Externý import má však aj dve nevýhody:

  1. Na vykonávanie externých importov musíme mať internetové pripojenie
  2. Pingovanie URL adresy na získanie dátového setu je pomerne časovo náročné, čo znamená, že vykonávanie externých importov spomalí rýchlosť nášho Pythonu kódu.
