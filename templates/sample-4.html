Getter/Setter a @Property v Pythone

V objektovo orientovaných jazykoch sa na získavanie a aktualizáciu údajov používajú takzvané "getters" a "setters" metódy. Aby som to nekomplikoval tak ponechám vyššie uvedné pojmy v anglickej verzii. Getter metóda získava aktuálnu hodnotu atribútu objektu, zatiaľ čo setter metóda hodnotu atribútu objektu mení. V tejto ukážke sa pozrieme na niekoľko príkladov ako getters a setters metódy použiť.

Getter je metóda, ktoré sa používa v objektovo orientovanom programovaní (OOP) na prístup k privátnym atribútom daného classu.

Setter je metóda, ktorá sa používa na nastavenie hodnoty objektu daného classu. V objektovo orientovanom programovaní je možnosť nastavovať alebo meniť hodnotu privátnych atribútov v danom classe veľmi užitočná.

Vo všeobecnosti getters a setters metódy sa v objektovo orientovanom programovaní používajú na takzvané zapuzdrenie údajov.

Funkcionalita metód "getters" a "setters" použitím bežnej funkcie

V ukážke nižšie môžeme vidieť, ako využiť funkcionalitu metód getters a setters použitím bežnej funkcie: 

class YearOfBirth:   
    def __init__(self, year_of_birth = 0):   
         self._year_of_birth = year_of_birth   
         
    # using the getter method   
    def get_year_of_birth(self):   
        return self._year_of_birth   

    # using the setter method   
    def set_year_of_birth(self, x):   
        self._year_of_birth = x   
    
Alessanda = YearOfBirth()   
    
#using the setter function  
Alessanda.set_year_of_birth(1981)   
    
# using the getter function  
print(Alessanda.get_year_of_birth())   
    
print(Alessanda._year_of_birth)
# Output
1981
1981

Na dosiahnutie tejto funckionality, funkcie get_year_of_birth() a set_year_of_birth() vo vyššie uvedenom kóde fungujú ako štandardné funkcie nevzťahujúce sa na "getters" a "setters" funkcionalitu. Python však obsahuje jedinečnú funkciu s názvom property().

Použitie property() funkcie na dosiahnutie funkcionality metód "getters" a "setters"

Funkcia property() je v Pythone vstavanou funkciou na vytváranie a vracanie property objektov. Na property objekty môžeme použiť tri metódy a to sú: getter(), setter() a delete(). V Pythone funkcia property() berie štyri argumenty: property(fget, fset, fdel, doc). Na získanie hodnoty atribútu sa používa funkcia fget. Na nastavenie hodnoty atribútu slúži funkcia fset. Na vymazanie hodnoty atribútu sa používa funkcia fdel.

V ukážke nižšie môžeme vidieť, ako môžeme použiť funkciu property() na dosiahnutie funkcionality metód getters a setters: 

class YearOfBirth:
    def __init__(self):
        self._year_of_birth = 0

    # using the get function
    def get_year_of_birth(self):
        print("getter called")
        return self._year_of_birth

    # using the set function
    def set_year_of_birth(self, x):
        print("setter called")
        self._year_of_birth = x

    # using the del function
    def del_year_of_birth(self):
        del self._year_of_birth

    year_of_birth = property(get_year_of_birth, set_year_of_birth, del_year_of_birth)  
    
Alessanda = YearOfBirth() 
    
Alessanda.year_of_birth = 1981
    
print(Alessanda.year_of_birth)
# Output
setter called
getter called
1981

Vo vyššie uvedenom programe vidíme výstup, ktorý sa skladá z troch riadkov. Program zavolá "setter" metódu set_year_of_birth() a následne "getter" metódu get_year_of_birth() ako vidíme v prvých dvoch riadkoch nášho výpisu.. Vo výsledku je teda year_of_birth objekt, ktorý pomáha v zabezpečení privátneho prístupu k premennej, čo je podstata tejto funkcionality.

Použitie @property dekorátorov na dosiahnutie funkcionality metód "getters" a "setters"

Na dosiahnutie funkcionality metód getters a setters sme v predchádzajúcej metóde použili funkciu property(). Ako však už bolo spomenuté v tomto článku, metódy getters a setters sa používajú aj na validáciu získavania a nastavovania hodnôt atribútov. Ďalšou metódou na vynútenie property() funkcie je použitie dekorátora. @property je jedným z vstavaných dekorátorov v Pythone. Primárnym cieľom každého dekorátora je upraviť metódy alebo atribúty classu tak, aby používateľ classu nepotreboval meniť svoj kód.

Pozrime sa na príklad, aby sme pochopili, ako môžeme property() dekorátory použiť na dosiahnutie funkcionality metód getters a setters: 

class YearOfBirth:   
    def __init__(self):   
        self._year_of_birth = 0  
        
    # using the getter function   
    @property  
    def year_of_birth(self):   
        print("getter called")   
        return self._year_of_birth   
        
    #now, using the setter function   
    @year_of_birth.setter   
    def year_of_birth(self, x):   
        if(x < 1850):   
           raise ValueError("It is not possible you are alive :D")   
        print("setter called")   
        self._year_of_birth = x 
    
Alessanda = YearOfBirth()   
    
Alessanda.year_of_birth = 1981 
    
print(Alessanda.year_of_birth)
# Output
setter called
getter called
1981

Predchádzajúci kód demonštruje, ako použiť dekorátor @property na vytváranie getters a setters v Pythone. Vyššie uvedený príklad slúži ako overovací kód, kde nám pri pokuse inicializovať rok narodenia s hodnotou menšou ako 1850 program vyhodí ValueError.

Nastavenie atribútov na súkromné

Teraz sa pozrime na to, ako vykresliť metódy ako súkromné, aby externe volané funkcie nemohli manipulovať s premennými v nich. Manipulovať s nimi bolo možné iba pomocou funkcií, ktoré boli definované v rámci classu. Súkromnými sa stanú pridaním prefixu dvoch podčiarkovníkov: 

class YearOfBirth:  
    def __init__(self, year_of_birth=1850):  
        self._year_of_birth = year_of_birth

# make the getter method  
    def get_year_of_birth(self):  
        return self.__year_of_birth 

# make the setter method  
    def set_year_of_birth(self, x):  
        self.__year_of_birth = x 

Alessanda = YearOfBirth() 

print(Alessanda._year_of_birth)  

# Before using setter  
print(Alessanda.get_year_of_birth()) 

# After using setter  
Alessanda.set_year_of_birth(1981)  
print(Alessanda._year_of_birth)
# Output
1850
Čítanie hodnôt zo súkromných metód

Na čítanie hodnôt zo súkromných metód musíme použiť metódu getter. Bez použitia metódy getter nemôžeme použiť property metódu na prístup k hodnotám súkromných atribútov. Vezmime si príklad na pochopenie tejto metódy: 

# A program to read the value from private method  
class YearOfBirth:  
    def __init__(self, year_of_birth=1850):  
        self._year_of_birth = year_of_birth  
 
    @property  
    def year_birth(self):  
        return self.__year_of_birth  
 
    @year_birth.setter  
    def year_birth(self, x):  
        self.__year_of_birth = x  
  
Alessanda = YearOfBirth()  
print(Alessanda._year_of_birth)  
  
Alessanda.year_of_birth = 2020  
print(Alessanda.year_of_birth)
# Output
1850
2020

Zoberme si ďalší príklad na vytvorenie classu s objektmi. Vytvorili sme niekoľko objektov:

class Pizza:
    def __init__(self):
        self.ordered_pizza = "None"
Alessanda_order = Pizza()
Adriana_order = Pizza()

Namiesto toho, aby sme museli na týchto objektoch nastaviť property metódu sme priamo zostrojili dve metódy: get_order() a set_order(), vzhľadom na to, že prvá spomenutá možnosť by nebola efektívna metóda: 

class Pizza:
    def __init__(self):
        self.ordered_pizza = "None"

    def get_order(self):
        return self.ordered_pizza

    def set_order(self, ordered_pizza):
        self.ordered_pizza = ordered_pizza

Alessanda_order = Pizza()
Adriana_order = Pizza()

Alessanda_order.set_order("Prosciutto")
Adriana_order.set_order("Diavola")

print(Alessanda_order.ordered_pizza)
print(Adriana_order.ordered_pizza)
# Output
Prosciutto
Diavola
