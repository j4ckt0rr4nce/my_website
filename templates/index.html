Django Web/App Development

Vývoj webových stránok alebo aplikácii vo frameworku Django. Django je high-level Python webový framework, ktorý podporuje rýchly vývoj a čistý, pragmatický dizajn. Dominuje predovšetkým svojou jednoduchosťou, flexibilitou, spoľahlivosťou a širokou škálou rozšíriteľnosti od MVP produktov až po škálovateľné podnikové produkty.

Flask Web/App Development

Vývoj webových aplikácii vo frameworku Flask. Flask je mikro webový framework napísaný v programovacom jazyku Python. Flask závisí od template enginu Jinja a sady nástrojov Werkzeug WSGI. Prednostné vlastnosti frameworku Flask: lightweight, open source a ponúka za pomoci minima kódu vývoj skvelých aplikácií.

Python Dátová Analýza

Dátová Analýza v jazyku Python za použitia flexibilných knižníc ako NumPy, Pandas, Matplotlib, či Seaborn. Efektívne spracovanie dátových setov formou získania, čistenia, párovania a kombinovania dát. Bez dát to v dnešnej dobe nejde a preto stále viac spoločností sleduje, analyzuje a využíva náhľady, ktoré ich vedú k lepším rozhodnutiam.

Front-end Web Development

Ľahká a príjemná interakcia návštevníkov s webovými stránkami je pri tvorbe dizajnu prioritou. Tvorba dizajnu zahŕňa nasledovné: Optimalizácia užívateľského dojmu, Využitie HTML, JavaScript a CSS na uvedenie konceptov do živej podoby, Vývoj užívateľského rozhrania a v neposlednom rade Implementácia responzívneho dizajnu.

Web Scraping / Crawling

Web scraping/crawling je technika automatického prístupu a extrakcie veľkého množstva informácií z webových stránok, čo môže ušetriť obrovské množstvo času a úsilia a to dosiahneme využitím obľúbenej Python knižnice Beautiful Soup, ktorá poskytuje idiomatické spôsoby navigácie, vyhľadávania a úpravy rozloženia potrebných údajov.

Python Basic Scripting

Python je skriptovací, univerzálny, vysokoúrovňový a interpretovaný programovací jazyk. Vzhľadom na to môže byť určený na vykonávanie úloh na základe automatizácie opakovaných úloh a to tak, že zahŕňa rovnaké typy krokov pri implementácii postupu alebo programu. Skracuje čas a ďalej znižuje náklady. Automatizácia úloh má veľmi širokú oblasť využitia.
